# Snippets

**简介**
为`vim`插件`coc-snippet`提供代码片段

`coc-snippet`代码片段的默认存放路径为:`$VIMCONFIG/coc/ultisnips`, 实例路径为`/home/haoming/.config/coc/ultisnips`


**自定义代码片段**

执行命令`:CocCommand snippets.editSnippets`即可打开代码自定义编辑器.
